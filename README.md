# 艺术拼图(PuzzleBID)介绍
## 1. 介绍
艺术拼图(PuzzleBID)是野狼队参与2018张江黑客马拉松的一个项目。它是一个部署在BOTTOS链上的艺术作品拼图交易游戏， 旨在为原创数字绘画作品重新发现价值，确定市场定价，并确立版权保护确权体系。
